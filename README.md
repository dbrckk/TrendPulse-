# TrendPulse

TrendPulse is a mobile-first Amazon deal website focused on the US market. It highlights budget-friendly finds, gift ideas, cheap tech, and popular Amazon products through category pages and individual product detail pages.

## Core Direction

- Mobile-first deal browsing
- US Amazon market prioritized
- Affiliate-driven product detail flow
- Static HTML pages with centralized product data
- Shared UI rendering through browser-side JavaScript
